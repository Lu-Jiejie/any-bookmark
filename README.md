# Any Bookmark · 万能收藏夹

为**任何没有收藏功能的网站**添加一个按域名分组的收藏夹面板的油猴脚本。

~~说的就是你 yande.re~~

基于[vite-plugin-monkey](https://github.com/lisonge/vite-plugin-monkey)。

## 安装

1. 先安装 [Tampermonkey](https://www.tampermonkey.net/)
2. 安装[脚本](https://github.com/Lu-Jiejie/any-bookmark/raw/gh-pages/any-bookmark.user.js)
